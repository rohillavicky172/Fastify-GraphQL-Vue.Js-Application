# Build a Full Stack MongoDB, Fastify, GraphQL and Vue.js application

> A Node.js project

## Build Setup

### Server
```bash
cd server

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run start
```

### Client
```bash
cd client

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve
```
